# Gerenciador de Clientes - Python + SQLite

Este projeto é um CRUD em Python utilizando SQLite para gerenciar clientes de um banco fictício.

## Funcionalidades
- Adicionar clientes com nome, saldo e CPF (com verificação de duplicidade)
- Atualizar saldo dos clientes e mostrar o saldo atualizado
- Deletar clientes pelo ID (com verificação de existência)
- Listar todos os clientes cadastrados
- Menu interativo no terminal para facilitar a utilização

## Tecnologias
- Python 3
- SQLite

## Como usar
1. Clone este repositório
2. Execute o arquivo `main.py`
3. Utilize o menu interativo para gerenciar clientes
